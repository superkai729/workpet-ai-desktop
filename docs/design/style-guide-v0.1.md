# 工位小宠视觉规范 v0.1

## 风格定位

温柔像素办公风：像素宠物是主角，UI 保持轻量、清晰、低打扰，适合长期悬浮在 Windows / macOS 桌面。

## 色彩

- 奶白：`#fff9f0`
- 暖灰：`#f3efe7`
- 墨灰：`#2f3136`
- 柔和橙：`#f4a261`
- 薄荷绿：`#8ecdb7`
- 柔蓝：`#8ab6d6`

## UI 原则

- 桌宠本体无窗口边框，透明背景，始终置顶。
- 菜单使用轻边框、低阴影、8px 圆角。
- 气泡短小，低频出现。
- 工具窗口保持办公工具感，不做过度游戏化。
- 宠物动作优先保证可爱、清晰、低打扰。

## MVP 动作

| 状态 | 用途 |
| --- | --- |
| 待机 | 默认状态 |
| 开心 | 摸摸后触发 |
| 吃饭 | 喂食后触发 |
| 睡觉 | 用户主动触发 |
| 思考 | 翻译 / 问答时触发 |
| 行走 | 后续加入 |
